Аудио плеер

Музыкальный плеер, позволяющий по очереди проигрывать музыкальные треки или перелистывать их кликами по кнопкам. Каждому музыкальному треку соответствует определённое фоновое изображение. Адаптивная верстка для экранов 320px и выше.
